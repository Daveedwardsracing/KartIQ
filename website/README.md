# RaceIQ Website

This folder contains a standalone multi-page marketing website for the RaceIQ product.

## Files

- `index.html` - homepage
- `platform.html` - platform overview
- `pricing.html` - pricing page
- `about.html` - business and product story
- `contact.html` - contact and demo page
- `styles.css` - shared styling for the site

## Preview

Open `index.html` directly in a browser for a quick preview and navigate between pages.

## Notes

- The contact links currently use placeholder email and phone values.
- Example pricing is currently set to `GBP 20/mo` or `GBP 200/year`.
- The site is designed as a commercial product website for selling RaceIQ to other teams.
- It can be expanded later with screenshots, testimonials, a lead form, or migrated into Next.js if needed.
